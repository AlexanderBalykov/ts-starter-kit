Starter Kit for new projects 
==== 

`Typescript` as language   
`SCSS` modules as styling engine  
`Webpack` as bundler  
`Jest` as test engine  
`React` as render library  

The project almost empty but enough for run and add something

**Happy debugging!**

Install
---
```
npm i
```

Run 
----
```
npm start
```

Run Tests
---
```
npm test
```

Look 
---
https://localhost:4001
